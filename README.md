COS2614 Assignment 1 – Contemporary Programming Concepts

Overview

This repository contains my solutions for COS2614 Assignment 1 completed during my Bachelor of Science in Computing studies at the University of South Africa (UNISA).

The assignment introduces practical software development using C++ and the Qt Framework, with a focus on graphical user interface development, object-oriented programming, class design, inheritance, and software implementation based on UML specifications.

---

Technologies Used

- C++
- Qt Creator
- Qt Widgets
- Qt Core
- Object-Oriented Programming (OOP)

---

Assignment Objectives

The assignment consists of two independent programming tasks.


Question 1 – Username and Password Generator

Develop a Qt GUI application that:

- Accepts a user's full name using a `QInputDialog`
- Generates a username according to predefined rules
- Generates a random initial password
- Displays the generated credentials using a `QMessageBox`
- Handles invalid user input

## Concepts Demonstrated

- GUI Programming
- QString manipulation
- Functions
- Conditional statements
- Loops
- Random number generation
- User input validation

---

Question 2 – Product and Vendor Management System

Develop a console application implementing classes from a supplied UML diagram.

The application allows users to:

- Create products
- Create vendors
- Associate vendors with products
- Display formatted product information

### Classes Implemented

- Item
- Vendor
- StoreItem

### Concepts Demonstrated

- Object-Oriented Programming
- Encapsulation
- Inheritance
- Constructors
- Member functions
- Dynamic memory allocation
- Pointers
- Console input/output

How to Run

## Requirements

- Qt Creator
- Qt 5 (or later)
- C++ Compiler (MinGW or MSVC)

---

### Running Question 1

1. Open **Question1-Username-Generator** in Qt Creator.
2. Build the project.
3. Run the application.
4. Enter the user's full name when prompted.

---

### Running Question 2

1. Open **Question2-Product-Vendor-System** in Qt Creator.
2. Build the project.
3. Run the console application.
4. Enter the product and vendor details when prompted.
